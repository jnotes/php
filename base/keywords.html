关键字

所谓的关键字就是被语言本身使用于实现一些基本操作的字，这些字不能用于其他用途。由于PHP变量的特性，所有的变量都是由$符号开头，大大方便了编写程序时对关键字的躲避。

用于数据类型定义的关键字

  • int
  • float
  • string
  • bool
  • class
  • function
  • interface
  • object
  • array

用于流程控制结构的关键字

  • if
  • else
  • elseif
  • do
  • while
  • for
  • break
  • continue
  • switch
  • case

用于设置存储的关键字

  • static
  • global

其他

  • try
  • catch
